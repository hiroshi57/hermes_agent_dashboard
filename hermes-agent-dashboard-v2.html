サンドボックスの注意

プロファイルは状態を分離しますが、ファイルシステムのサンドボックスではありません。local バックエンドではユーザー権限と同じアクセス範囲を持ちます。隔離が必要な場合は docker バックエンドへ切り替え、作業ディレクトリは terminal.cwd を絶対パスで固定してください。
