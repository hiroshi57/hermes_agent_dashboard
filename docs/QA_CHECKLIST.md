## 2. グローバル機能

### コマンドパレット（⌘K / Ctrl+K）
- [ ] `⌘K`（Mac）/ `Ctrl+K`（Win/Linux）でパレットが開く
- [ ] ページ名を入力すると候補が絞り込まれる
- [ ] `↑`/`↓` で選択移動、`Enter` で遷移、`Esc` で閉じる
- [ ] 候補ラベル・グループバッジが現在の言語に追従する

### キーボードナビ
- [ ] `j` / `k` で前後のページに移動
- [ ] `g` + キー（例 `g`→`k` で Kanban）で直接ジャンプ
- [ ] `/` で現在ページの検索ボックスにフォーカス
- [ ] `?` でショートカットヘルプのトーストが出る

### 言語切替（i18n）
- [ ] トップバーの言語トグルで JA ⇄ EN が切り替わる
- [ ] ナビ・見出し・ページタイトルが翻訳される

### テーマ切替
- [ ] テーマボタンでポップアップが開き、6テーマ（Hermes Teal / Midnight / Ember / Mono / Cyberpunk / Rosé）を選べる
- [ ] 選択で全ページの配色が即時に変わる

## 3. ページ別インタラクション
